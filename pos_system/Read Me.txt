## ðŸ› Troubleshooting

### Camera Scanner Not Working
```
Problem: Camera permission denied
Solution: 
  - Check browser camera permissions
  - Go to browser Settings â†’ Privacy
  - Enable camera for website
  - Reload page

Problem: No barcode detected
Solution:
  - Ensure good lighting
  - Hold barcode steady
  - Zoom in if barcode small
  - Try different angle
  - Use Image upload instead

Problem: Camera not available
Solution:
  - Check device has camera
  - Grant camera permissions
  - Use mobile device
  - Try different browser
```

### Image Upload Issues
```
Problem: "No barcode found in image"
Solution:
  - Image must contain visible barcode
  - Try different image
  - Ensure high contrast
  - Increase image resolution
  - Try camera scanner instead

Problem: File too large
Solution:
  - Maximum file size: 10MB
  - Compress image
  - Take new photo
```

### Barcode Not Scanning
```
Problem: Product not found after scan
Solution:
  - Verify product added to inventory
  - Check barcode number matches
  - Ensure product quantity > 0
  - Check for typos in barcode

Problem: Wrong product appears
Solution:
  - Verify barcode number is correct
  - Check for duplicate barcodes
  - Delete incorrect product
  - Re-scan or re-enter barcode
```

### Data Not Saving
```
Problem: Data lost after refresh
Solution:
  - Check localStorage not disabled
  - Clear browser cache
  - Try incognito mode
  - Check browser storage limit

Problem: Cannot import data
Solution:
  - Verify JSON file format
  - Check file corruption
  - Try exporting/importing small set
```
